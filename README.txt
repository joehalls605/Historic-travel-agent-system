


SERVER AND MONGODB SETUP:

1. Set up MongoDB Atlas
Create a database
Allow IP address in Network Access

2. Install Required Packages
npm install express mongoose cors dotenv mongodb

3. Add Your `.env` File
Create a `.env` file with your MongoDB connection string:
MONGO_URI=your_mongo_connection_string

4. Set up the Backend Server
Use code in server.js as reference.

5. Insert Sample Data
Create a file called `insertBookings.js`
Use code as reference in this project.

6. Run Server
node server.js
